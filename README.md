# models-desercion
Este repositorio contiene los modelos desarrollados para la tesis, cuyo objetivo es analizar y predecir la deserción estudiantil en la carrera de Sistemas de Información de la Universidad Central del Ecuador.

El proyecto se basa en el uso de variables sociodemográficas para:

Estimar el porcentaje de deserción estudiantil.

Identificar los factores sociodemográficos con mayor incidencia en la deserción.

Construir y evaluar modelos predictivos de clasificación que permitan anticipar el riesgo de abandono académico.

Los modelos implementados incluyen técnicas de machine learning supervisado, tales como Regresión Logística, Árboles de Decisión, Random Forest, Gradient Boosting y XGBoost, evaluadas mediante métricas apropiadas para problemas con posible desbalance de clases.

Este repositorio tiene como finalidad garantizar la reproducibilidad del análisis, apoyar la toma de decisiones académicas y servir como base para futuras investigaciones relacionadas con la permanencia estudiantil.
